RadarRelay is a decentrazlied exchange that uses the 0x protocol, or otherwise reffered to as a 0x Relayer. RadarRelay is essentially a matching and aggregration engine for orders or trades placed on the 0x protocol. There are many 0x relayers currently in development. However, RadarRelay currently seems to be the only one fully functional on the main Ethereum network with a varity if tokens to swap in-between.

Judging from 0xtrades.info and 0xtracker.com, RadarRelay seems to be doing ~95%, if not almost all of the volume on the 0x protocol. There are some occasional direct swaps not being faciliated by relayers, but right now these sites could temporarly be re-named RadarRelay Explorer.

When you get started with RadarRelay, you'll see the need for W-ETH. W-ETH is "wrapped ETH", an ERC20 token on Ethereum that represents real Ether. When you "wrap" ETH, you aren't actually wrapping anything. What you're doing is converting ETH for an equal token called W-ETH via a smart contract. To get regular ETH back, you "unwrap" it, aka just trade it back for plain ETH. Suprisingly enough, ETH isn't ERC20 compliant, and having W-ETH allows you to have ETH value in an ERC20 token swap.
